MagicPhotos 1.0.0

Developer: Oleg Derevenetz <oleg.derevenetz@gmail.com>

MagicPhotos is an application for Blackberry 10 devices that lets you quickly and easily give photos a dramatic look by applying different effects while keeping your chosen details untouched.

If you enjoyed this application, please use button below to recommend this app to your friends.

If you have any questions regarding this application, you can contact application developer on this website: http://magicphotos.sourceforge.net/.

Switching between modes

Use swipe gestures on mode selection screen to switch between modes.

Decolorize mode

Decolorize mode lets you give photos a dramatic look by converting them to black and white, while keeping chosen details in color. This effect draws the viewer's attention to the colored areas, creating striking images. Use the following buttons in this mode:

button to move and resize image

button to bring back original color of the part of an image

button to make part of an image black and white

Sketch mode

Sketch mode lets you convert your photos to sketches, while keeping chosen details untouched. You will have an opportunity to control sketch generation process. Use the following buttons in this mode:

button to move and resize image

button to bring back original look of the part of an image

button to convert part of an image to sketch

Recolor mode

Recolor mode lets you recolor your photos to create amazing new artwork and effects. Use the following buttons in this mode:

button to move and resize image

button to bring back original color of the part of an image

button to apply selected color to the part of an image

button to select a desired color from the palette
